Fluxvod Mobile

Developed an Android application for streaming movies, featuring an intuitive interface and user-friendly navigation. The app provides access to a vast library of movies and TV shows, with options to filter content by genre, popularity, and new releases. Special attention was given to performance to ensure smooth video playback. Additionally, a user account system was implemented to save viewing history, bookmarks, and personalized recommendations based on user preferences.

I also developed an Android TV application that includes features such as user authentication, subscription management, and in-app purchases. This enables users to subscribe to premium content, access special offers, and seamlessly integrate additional services through the in-app purchase system.
